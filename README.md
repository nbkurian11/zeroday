# ZeroDay 

The day your debt hits zero.

ZeroDay is a debt tracking web app that helps you visualize your debt, prioritize payoff using the avalanche method, and see exactly how many months until you're debt free.

## Features
- Add and track multiple debts
- Automatically sorted by interest rate (avalanche method)
- Payoff months calculator for each debt
- Total debt overview
- Delete debts when paid off

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS/JavaScript
- Deployed on Render

## Live Demo
https://zeroday-akem.onrender.com

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
